# ElementsOS v1

**ElementsOS v1** is a fully interactive HTML5 reference and learning platform built with **pure HTML, CSS, and vanilla JavaScript**. It demonstrates **over 100 HTML elements** through live examples, interactive components, and real-world patterns. This project is designed for students, beginners, and front-end developers to learn, explore, and practice web development fundamentals.

---

## Features

- **Comprehensive HTML Reference**: Covers 112 HTML elements across typography, forms, tables, media, semantic elements, interactive components, charts, and pricing layouts.
- **Interactive UI Components**: Modals, carousels, drag-and-drop kanban boards, dynamic tags, toast notifications, live clocks, and counters.
- **Advanced Visuals**: Canvas particle systems, SVG animations, animated donut charts, and scroll-reveal effects.
- **Accessibility First**: ARIA labels, semantic HTML structure, keyboard navigation, and color-independent information design.
- **Multiple Themes**: Dark, Light, and Cyberpunk modes.
- **Progressive Enhancement**: Works without JavaScript, enhanced with interactivity.
- **Clean & Maintainable Code**: Semantic HTML, CSS variables, responsive grid and flexbox layouts, well-commented JavaScript.

---

## Learning Benefits

- Study HTML semantics and accessibility in context.
- Explore real-world **form validation** and dynamic content patterns.
- Learn **DOM manipulation, event handling, and keyboard shortcuts** without frameworks.
- Understand how to build interactive components and advanced CSS/JS effects with zero dependencies.
